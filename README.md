## About

Based in Seoul. I write digital forensic tooling — filesystem parsers
for NTFS, exFAT, APFS, ext4. Windows artifact decoders (registry, EVTX,
prefetch, LNK, MFT). Browser SQLite extractors. Pipelines that move
evidence from raw bytes to analyst tables.

Currently curious about storage formats, parallel parsing, forensic data
models, and the slow craft of reading byte layouts from specs.
